## 📖 项目简介

**Fast Info** 是一个面向技术爱好者和开发者的技术情报分析平台。我们从全球优质数据源聚合技术资讯，并提供 AI 智能分析，帮助你高效获取有价值的信息。

### 🎯 产品定位

我们**不是**简单的内容转载平台，而是：
- ✅ 技术情报分析平台
- ✅ AI 驱动的原创分析
- ✅ 趋势发现和预测
- ✅ 中文友好的技术摘要

### ✨ 核心功能

- **🌐 多源聚合**：整合 Hacker News、GitHub Trending、Dev.to、Product Hunt、arXiv、AIBase 等 6+ 个优质数据源
- **🤖 AI 智能分析**：自动生成技术摘要和深度分析
- **📊 趋势预测**：基于多维数据的热度评分和趋势预测
- **🏷️ 智能分类**：自动分类为科技、开发者、开源、学术、产品等类别
- **🔍 搜索筛选**：强大的搜索和筛选功能
- **📱 响应式设计**：完美支持桌面和移动端

## 💡 为什么选择 Fast Info？

### 对比传统聚合平台

| 特性 | 传统聚合 | Fast Info |
|------|----------|-----------|
| 内容展示 | 简单转载 | 原创分析 ✨ |
| 中文支持 | 需要 VPN | AI 中文摘要 ✨ |
| 趋势预测 | ❌ | ✅ 多维评分 ✨ |
| 数据质量 | 参差不齐 | 智能筛选 ✨ |
| 法律合规 | 可能侵权 | 严格合规 ✨ |

### 核心优势

1. **🚫 无需 VPN**：AI 生成中文摘要，无需访问原网站也能快速了解
2. **🔮 早期发现**：趋势预测算法帮你在项目爆火前发现机会
3. **📚 深度分析**：不只是标题党，提供有价值的技术解读
4. **⚖️ 合法合规**：严格遵守版权法，明确标注来源

## 🛠️ 技术栈

> 注：本 repo 仅用于项目介绍和用户反馈，不包含源代码

**前端**：Vue 3 + Vite + TailwindCSS + Geist 字体

**后端**：Node.js + Express + Supabase (PostgreSQL)

**AI**：Deepseek API

**爬虫**：Axios + Cheerio + Puppeteer + node-cron

**部署**：Vercel (前端) + Railway (后端)

## ❓ 常见问题

### Q1: Fast Info 是免费的吗？
**A**: 目前完全免费。未来可能推出付费功能（如 AI 深度分析），但基础功能会保持免费。

### Q2: 内容从哪里来？
**A**: 我们从 Hacker News、GitHub、Dev.to 等公开数据源获取信息，通过官方 API 或公开页面。

### Q3: AI 分析准确吗？
**A**: AI 分析仅供参考，可能存在错误。重要决策前请查看原文。

### Q4: 可以转载 Fast Info 的内容吗？
**A**: 原文内容版权归原作者，请访问原链接。AI 生成的分析可以引用，但请注明来源。

### Q5: 为什么看不到原文？
**A**: 我们不抓取完整原文。点击文章卡片会跳转到原网站。AI 摘要可以帮你快速了解内容。

### Q6: 支持哪些语言？
**A**: 目前主要支持中文界面，但聚合的是全球英文资讯。AI 会生成中文摘要。
